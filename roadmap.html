API Test Generator · v1.0.87

Генерация тестов из
OpenAPI-спецификации

Генерация тестов
OpenAPI Parser
Разбор спецификаций OpenAPI 3.x и Swagger 2.0 в форматах JSON и YAML
Schema-Aware Assertions
Автовыбор типа проверки по типу поля, формату (uuid, email, date-time), enum-значениям и числовым ограничениям
Позитивные тесты
Генерация happy-path тестов с корректными данными запроса для каждого эндпоинта
Негативные тесты
Автогенерация на нарушение ограничений: enum, min/max, minLength/maxLength, неверный тип, 401/404
Вывод в pytest
Python-тесты с requests, поддержка функций и классов
Вывод в REST Assured
Java-тесты с JUnit 5 и Hamcrest-матчерами
Вывод в Postman
Postman Collection v2.1 с pm.test() скриптами и переменными окружения
Выборочная генерация
Выбор эндпоинтов через дерево с чекбоксами и быстрый выбор всех сразу
Загрузка по URL
Загрузка спека по ссылке с авторизацией и авто-обнаружением пути (Swagger UI, /v3/api-docs)
ZIP-архив
Автоматическая упаковка всех сгенерированных файлов в ZIP
Производительность
9 000 тестов в секунду — генерация большой спецификации мгновенно
HTTP Клиент
Отправка запросов
Полный HTTP-клиент: GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
Параметры запроса
Редактирование path params, query params, заголовков и тела в табличном виде
Просмотр ответа
Pretty (JSON с подсветкой), Raw, Headers — с временем и размером ответа
Pre-request скрипты
JavaScript-код до отправки запроса — для подготовки данных и токенов
Test-скрипты
JavaScript после ответа: test(name, cond) и полный pm.test / pm.expect API
QuickJS движок
Встроенный JS-движок в Desktop — без Python и Node.js
Коллекции и переменные
Коллекции запросов
Создание, редактирование и организация сохранённых HTTP-запросов
Переменные коллекции
{{baseUrl}}, {{token}} и любые другие с областью видимости коллекции
Глобальные переменные
Переменные доступные одновременно во всех коллекциях
JSONPath-извлечение
Извлечение $.data.id из ответа и сохранение в переменные для цепочек запросов
Библиотека сниппетов
Готовые JS-сниппеты для типовых проверок — вставка одним кликом
Экспорт коллекций
Конвертация коллекции в готовый pytest или REST Assured файл
Collection Runner
Последовательный запуск
Выполнение всех запросов коллекции по порядку в автоматическом режиме
Прогресс и результаты
Прогресс-бар и таблица: метод, статус, время, pass/fail по каждому запросу
Передача переменных
Значения из JSONPath в одном запросе доступны в следующих — цепочки
EVA — Анализ качества тестов
Oracle Depth L0–L6
Шкала глубины проверок: от «нет assertions» (L0) до «бизнес-логика» (L6)
Грейд качества S–F
Итоговая оценка набора тестов: S (90+), A, B, C, D, F
Анализ файла / папки
Разбор .py или .java файла или целой папки с агрегированным результатом
Детектирование anti-patterns
Thread.sleep, пустые catch-блоки, захардкоженные credentials, copy-paste тесты
Рекомендации
Конкретные советы по улучшению assertions на основе найденных проблем
Статический анализ
Анализ без запуска тестов — только по исходному коду
